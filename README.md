# ☕ Café Atlas – Agadir

A modern, premium restaurant & café website for **Café Atlas**, located in Agadir, Morocco.

## ✨ Features

- **5 Pages** — Accueil, Menu, Galerie, À Propos, Contact
- **Mobile-First Design** — Responsive across all devices with bottom navigation bar on mobile
- **Premium Aesthetic** — Luxury color palette (dark coffee, cream, gold), Playfair Display typography
- **Interactive Menu** — Category filters, dish cards with images, downloadable PDF menu
- **Photo Gallery** — Lightbox viewer with smooth animations
- **WhatsApp Integration** — Quick contact via WhatsApp, phone, and Google Maps
- **Smooth Animations** — Framer Motion page transitions and scroll-triggered reveals

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| **React 19** | UI library |
| **Vite 7** | Build tool & dev server |
| **Tailwind CSS 4** | Utility-first styling |
| **React Router 7** | Client-side routing |
| **Framer Motion 12** | Animations |
| **jsPDF** | PDF menu generation |
| **Lucide React** | Icons |

## 🚀 Getting Started

```bash
# Clone the repo
git clone https://github.com/Abdelouahedb/CafeAtlas-Agadir.git
cd CafeAtlas-Agadir

# Install dependencies
npm install

# Start dev server
npm run dev
```

## 📁 Project Structure

```
src/
├── components/       # Navbar, Footer, MobileBottomBar, Layout, SectionHeading
├── constants/        # Menu data, images, contact info
├── pages/            # Home, Menu, Gallery, About, Contact
├── index.css         # Tailwind config & custom styles
└── main.jsx          # App entry point
```
